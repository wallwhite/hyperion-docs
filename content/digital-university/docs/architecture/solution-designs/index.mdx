This section contains solution designs for features and technical implementations in the Digital University project.

Each solution design is paired with a test design.

## Creating a Solution Design

1. Create a new numbered folder (e.g., `002/`)
2. Use the solution design template
3. Create a paired test design
4. Add diagrams to support the design
5. Update this index page
