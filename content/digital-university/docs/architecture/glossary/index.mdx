# Glossary

This glossary provides translations and definitions for domain-specific terms used in the NPP Portal system.

## Organizational Structure

| Ukrainian         | English         | Definition                           |
| ----------------- | --------------- | ------------------------------------ |
| Інститут          | Institute       | Top-level organizational unit        |
| Факультет         | Faculty         | Division within an institute         |
| Кафедра           | Department      | Academic department within a faculty |
| Завідувач кафедри | Department Head | Head of an academic department       |
| Декан             | Dean            | Head of a faculty                    |

## Personnel

| Ukrainian                            | English             | Definition                         |
| ------------------------------------ | ------------------- | ---------------------------------- |
| Науково-педагогічний працівник (НПП) | Faculty Member      | University teaching/research staff |
| Викладач                             | Lecturer/Teacher    | Teaching staff member              |
| Асистент                             | Assistant           | Junior teaching position           |
| Доцент                               | Associate Professor | Mid-level academic rank            |
| Професор                             | Professor           | Senior academic rank               |
| Співавтор                            | Co-author           | Collaborator on an activity        |

| Ukrainian            | English         | Definition                                 |

## Activity & Reporting

| Ukrainian           | English        | Definition                                 |
| ------------------- | -------------- | ------------------------------------------ |
| Робота / Діяльність | Activity       | A reported work item                       |
| Тип роботи          | Activity Type  | Category from the dictionary               |

| Нормований час      | Norm Time      | Standard hours for activity type           |
| Рейтинг             | Rating         | Calculated performance score               |

## User Roles

| Ukrainian          | English       | Definition                                   |
| ------------------ | ------------- | -------------------------------------------- |
| Гість              | Guest         | Unauthenticated user (public access)         |
| Викладач           | Teacher       | Authenticated faculty member                 |
| Директор/Завідувач | Director      | Department/Faculty head with elevated access |
| Адміністратор      | Administrator | Full system access                           |

## API Terms

| Term          | Definition                                                        |
| ------------- | ----------------------------------------------------------------- |
| Access Token  | Short-lived JWT for API authentication (30 minutes)               |
| Refresh Token | Long-lived token for obtaining new access tokens (30 days)        |
| Soft Delete   | Marking record as inactive rather than permanent deletion         |
| Pagination    | Breaking large result sets into pages (`page`, `size` parameters) |
